# 🎓 ClassRoom Attendance App

A modern classroom attendance management application that simplifies attendance tracking for teachers and students. The system enables efficient attendance recording, monitoring, and management through a user-friendly interface.

## 🚀 Features

* 📋 Student attendance management
* ✅ Quick attendance marking
* 📊 Attendance statistics and reports
* 👨‍🏫 Teacher dashboard
* 👨‍🎓 Student records management
* 🔍 Search and filter functionality
* 📱 Responsive user interface
* 💾 Secure data storage

## ⚙️ Installation

### Prerequisites

* Node.js (v18 or later)
* npm or yarn
* MongoDB

### Clone the Repository

```bash
git clone https://github.com/Eshwar-Tamminidi/ClassRoom_Attendance_App.git
cd ClassRoom_Attendance_App
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

### Run the Application

```bash
npm start
```

or

```bash
npm run dev
```
